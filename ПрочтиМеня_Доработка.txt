Задача - доработать функции в языке Basic Plus для Clev3r, добавив возможность задавать возвращаемое значение, через оператор return. А также возможность делать вызов функций в выражениях, когда вместо функции используется вычисленное в ней возвращаемое значение. 
Решение: Дорабатываем препроцессор. Добавляем еще один предварительный проход по тексту программы (или дорабатываем текуший первый проход препроцессора), а именно: в определение функции разрешаем использовать зарезервированные операторы number и string после оператора Function и перед именем функции для задания типа возвращаемого значения функции. При этом осуществляем подмену - убираем оператор типа и в качестве последнего аргумента функции (формального параметра) вставляем выходной аргумент с именем Ret_NameFunction с заданным перед именем функции типом. То есть, вместо:

Function number NameFinction(in number First, in string Second)   ' получаем:

Function NameFinction(in number First, in string Second, out number Ret_NameFunction)

При обнаружении вызова функции в выражении, вставляем перед строкой, содержащей выражение, строку с вызовом функции, при этом добавляем в список аргументов функции (фактических значений параметров) в качестве последнего параметра имя переменной в виде F_NameFunction. А в выражение, вместо вызова функции вставляем эту добавленную переменную F_NameFunction. То есть, вместо:

A1 = 10 + NameFinction(5, S) - A2	' Получаем

NameFinction(5, S, F_NameFunction)
A1 = 10 + F_NameFunction - A2

или 

If NameFinction(A1, "Text") = 5 Then			' Получим
NameFinction(A1, "Text", F_NameFunction)
If F_NameFunction = 5 Then

Остальной процесс трансляции остается неизменным.
